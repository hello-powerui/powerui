# PowerUI Managers

This repository contains the manager scripts for PowerUI Webflow integration. The scripts provide functionality for theme management, custom palettes, and various UI utilities.

## Usage

To use these scripts in your Webflow project, add the following script tag to your project's custom code:

```html
<script src="https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/managers.js"></script>
```

Replace `YOUR-USERNAME` and `YOUR-REPO-NAME` with your actual GitHub username and repository name.

## Features

- Theme Management
- Custom Color Palettes
- Neutral Palettes
- DOM Utilities
- Event Management
- Style Management
